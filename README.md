# react-hot-express

Hot reloading starter project with React and Express.

# Dependencies
- Webpack 4+
- Babel 7+
- React 16+
- React Hot Loader 4+
- Express 4+
- Jest + Enzyme
- Sass
- MiniCssExtractPlugin
- ESLint

# Commands
Development
```
yarn dev
```

Production
```
yarn build && yarn start
```

Test
```
yarn test
```
